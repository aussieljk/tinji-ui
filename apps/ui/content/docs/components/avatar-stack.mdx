## Usage

```tsx
import {
  AvatarStack,
  AvatarStackOverflow,
} from "@/components/ui/avatar-stack"
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar"
```

```tsx
<AvatarStack>
  <Avatar>
    <AvatarImage src="/avatars/01.png" alt="User avatar" />
    <AvatarFallback>JD</AvatarFallback>
  </Avatar>
  <Avatar>
    <AvatarFallback>ML</AvatarFallback>
  </Avatar>
  <AvatarStackOverflow>+5</AvatarStackOverflow>
</AvatarStack>
```

## API Reference

### AvatarStack

Container that overlaps its avatars using negative spacing and applies a `ring-2 ring-background` to each child avatar so they stay visually separated. Renders a `div` with `data-slot="avatar-stack"`.

### AvatarStackOverflow

An avatar-shaped indicator for the remaining count (e.g. `+5`). Composes the base `Avatar` with a muted background.
