## API Reference

These are custom components, most built on Base UI's `useRender` hook rather than direct Base UI wrappers.

### Heading

Renders an `h1`–`h6` element. The `level` prop controls the rendered tag, while `size` controls the visual scale independently.

| Prop     | Type                                          | Default | Description                                       |
| -------- | --------------------------------------------- | ------- | ------------------------------------------------- |
| `level`  | `1 \| 2 \| 3 \| 4 \| 5 \| 6`                  | `2`     | The semantic heading level (rendered tag h1–h6).  |
| `size`   | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl"` | `"xl"`  | The visual size of the heading.                   |
| `render` | `React.ReactElement`                          | -       | Render as a different element.                     |

### Text

Renders a paragraph by default. Use the `render` prop for inline `span` usage.

| Prop     | Type                                  | Default     | Description                          |
| -------- | ------------------------------------- | ----------- | ------------------------------------ |
| `size`   | `"xs" \| "sm" \| "md" \| "lg"`        | `"md"`      | The text size.                       |
| `color`  | `"default" \| "muted"`                | `"default"` | The text color token.                |
| `render` | `React.ReactElement`                  | -           | Render as a different element.       |

### Strong

Renders a `strong` element for important content.

### Em

Renders an `em` element for emphasized content.

### Code

Renders an inline `code` element with a muted background.

### Quote

Renders an inline `q` element for short quotations.

### Blockquote

Renders a `blockquote` element for block-level quotations.

### Link

Renders an anchor element. Use the `render` prop to render as a framework link.

| Prop     | Type                 | Default | Description                              |
| -------- | -------------------- | ------- | ---------------------------------------- |
| `render` | `React.ReactElement` | -       | Render as a different element (e.g. Link). |
