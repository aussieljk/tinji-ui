## Usage

```tsx
import {
  DataList,
  DataListItem,
  DataListLabel,
  DataListValue,
} from "@/components/ui/data-list"
```

```tsx
<DataList orientation="horizontal">
  <DataListItem>
    <DataListLabel>Status</DataListLabel>
    <DataListValue>Active</DataListValue>
  </DataListItem>
  <DataListItem>
    <DataListLabel>Name</DataListLabel>
    <DataListValue>Ada Lovelace</DataListValue>
  </DataListItem>
</DataList>
```

## API Reference

Renders semantic `<dl>` / `<dt>` / `<dd>` elements.

### DataList

Root component (`<dl>`). Supports an `orientation` prop.

| Prop          | Type                          | Default      | Description                                  |
| ------------- | ----------------------------- | ------------ | -------------------------------------------- |
| `orientation` | `"horizontal" \| "vertical"`  | `"vertical"` | Lays out label and value side by side or stacked |

### DataListItem

Wraps a single label/value pair (`<div>`).

### DataListLabel

The muted key (`<dt>`).

### DataListValue

The value (`<dd>`).
