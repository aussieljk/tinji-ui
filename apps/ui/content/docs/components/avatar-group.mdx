## Usage

```tsx
import {
  AvatarGroup,
  AvatarGroupOverflow,
} from "@/components/ui/avatar-group"
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar"
```

```tsx
<AvatarGroup>
  <Avatar>
    <AvatarImage src="/avatars/01.png" alt="User avatar" />
    <AvatarFallback>JD</AvatarFallback>
  </Avatar>
  <Avatar>
    <AvatarFallback>ML</AvatarFallback>
  </Avatar>
  <AvatarGroupOverflow>+3</AvatarGroupOverflow>
</AvatarGroup>
```

## API Reference

### AvatarGroup

Container that lays out avatars in a row with consistent spacing. Renders a `div` with `data-slot="avatar-group"`.

### AvatarGroupOverflow

An avatar-shaped indicator for the remaining count (e.g. `+3`). Composes the base `Avatar` with a muted background.
