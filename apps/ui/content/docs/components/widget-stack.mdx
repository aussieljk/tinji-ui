## Usage

```tsx
import {
  WidgetStack,
  WidgetStackItem,
} from "@/components/ui/widget-stack"
```

```tsx
<WidgetStack>
  <WidgetStackItem depth={0}>Today</WidgetStackItem>
  <WidgetStackItem depth={1}>Weather</WidgetStackItem>
  <WidgetStackItem depth={2}>Activity</WidgetStackItem>
</WidgetStack>
```

Items share the same grid cell and are offset by their `depth`. The front-most
card uses `depth={0}`; higher values sit further back, peeking out below with a
slight scale and opacity reduction.

## API Reference

This is a custom component using Base UI's `useRender` hook, not a direct Base UI wrapper.

### WidgetStack

Root container that layers its items in a single grid cell. Supports the
`render` prop.

### WidgetStackItem

A card in the stack. Supports the `render` prop.

| Prop     | Type                 | Default | Description                                                |
| -------- | -------------------- | ------- | ---------------------------------------------------------- |
| `depth`  | `number`             | `0`     | Stack depth; `0` is front-most, larger values sit behind   |
| `render` | `React.ReactElement` | -       | Render as a different element                              |
