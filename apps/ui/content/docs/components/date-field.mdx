## Usage

```tsx
import { DateField } from "@/components/ui/date-field"
```

```tsx
<DateField aria-label="Date of birth" />
```

Type into any segment to advance automatically to the next one. Use the arrow
keys to move between segments, and `Backspace` on an empty segment to step back.

## Controlled

```tsx
"use client"

import { useState } from "react"

import { DateField, type DateFieldValue } from "@/components/ui/date-field"

export function Example() {
  const [value, setValue] = useState<DateFieldValue>({
    day: "",
    month: "",
    year: "",
  })

  return (
    <DateField aria-label="Date" onValueChange={setValue} value={value} />
  )
}
```

## API Reference

This is a custom component built from native input segments.

### DateField

| Prop            | Type                                  | Default | Description                                       |
| --------------- | ------------------------------------- | ------- | ------------------------------------------------- |
| `value`         | `DateFieldValue`                      | -       | Controlled value (`{ day, month, year }`).        |
| `defaultValue`  | `DateFieldValue`                      | -       | Uncontrolled initial value.                       |
| `onValueChange` | `(value: DateFieldValue) => void`     | -       | Called whenever a segment changes.                |
| `disabled`      | `boolean`                             | `false` | Disables all segments.                            |
| `aria-label`    | `string`                              | -       | Accessible label for the field group.             |
