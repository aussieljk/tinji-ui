## API Reference

The Date Range Picker is a composition of the `<Calendar>` (with `mode="range"`) and `<Popover>` components. It works in both controlled and uncontrolled modes.

### DateRangePicker

| Prop             | Type                                          | Default              | Description                                       |
| ---------------- | --------------------------------------------- | -------------------- | ------------------------------------------------- |
| `value`          | `DateRange`                                   | -                    | The selected range (controlled).                  |
| `defaultValue`   | `DateRange`                                   | -                    | The initial range (uncontrolled).                 |
| `onValueChange`  | `(range: DateRange \| undefined) => void`     | -                    | Called when the selected range changes.           |
| `placeholder`    | `string`                                      | `"Pick a date range"`| Text shown when no range is selected.             |
| `formatStr`      | `string`                                      | `"LLL dd, y"`        | `date-fns` format string for the trigger label.   |
| `numberOfMonths` | `number`                                      | `2`                  | Number of months shown in the calendar.           |
| `disabled`       | `boolean`                                     | -                    | Disables the trigger button.                      |
| `open`           | `boolean`                                     | -                    | The open state of the popover (controlled).       |
| `defaultOpen`    | `boolean`                                     | `false`              | The initial open state (uncontrolled).            |
| `onOpenChange`   | `(open: boolean) => void`                     | -                    | Called when the popover open state changes.       |
