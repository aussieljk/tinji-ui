```tsx
  <RadioButtonGroupItem value="enterprise">Enterprise</RadioButtonGroupItem>

```

## API Reference

Built on Base UI's `RadioGroup` and `Radio` primitives. Each item is a button-like selectable box that highlights with a primary border and ring when checked (via the `data-checked` state).

### RadioButtonGroup

Root component. Styled wrapper for `RadioGroup.Root` from Base UI with flex column layout.

### RadioButtonGroupItem

An individual selectable button. Styled wrapper for `Radio.Root` from Base UI. Supports any children as the label content and reflects its state with `data-checked` / `data-disabled`.
