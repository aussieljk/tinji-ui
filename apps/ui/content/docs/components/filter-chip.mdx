## Usage

```tsx
import { FilterChip, FilterChipRemove } from "@/components/ui/filter-chip"
```

```tsx
<FilterChip selected={selected} onClick={() => setSelected((v) => !v)}>
  Active
  <FilterChipRemove onClick={() => setSelected(false)} />
</FilterChip>
```

## API Reference

### FilterChip

A rounded-full toggle button with a selected state. Uses Base UI's `useRender` hook.

| Prop       | Type                 | Default     | Description                                |
| ---------- | -------------------- | ----------- | ------------------------------------------ |
| `selected` | `boolean`            | -           | Whether the chip is selected               |
| `size`     | `"default" \| "sm"`  | `"default"` | Controls the chip size                     |
| `render`   | `React.ReactElement` | -           | Render as a different element              |

### FilterChipRemove

An icon button for dismissing the chip. Renders an `XIcon` by default.
